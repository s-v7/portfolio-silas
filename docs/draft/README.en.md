# Silas Vasconcelos Cruz

## About
Silas is a Technology & Information Security Advisor at CREA-PI based in Teresina, Piauí, Brasil. His work focuses on creating and optimizing technology systems that enhance security and efficiency within various domains.

## Projects
- **SIGEC v2 (Fiscal Intelligence Engine)**: Developed a comprehensive fiscal intelligence platform that indexes over 1.1 million ARTs on the SHA-256 blockchain. Implemented a Sentinel-2 to PostGIS pipeline for hydrological monitoring across 20 reservoirs utilizing NDWI. Additionally, created a FiscalBot on Telegram for automated alerts and integrated a NetworkX-based TSP routing system for optimizing resource distribution across 224 municipalities in Piauí. The tech stack includes Flask, FastAPI, Angular 17, PostgreSQL 17, PostGIS, and 8 systemd services in production.

- **SistemaCrea (Jakarta EE)**: Modernized an ERP solution using a multi-module EAR structure with Maven and integrated with e-MEC for 4,328 Institutions of Higher Education (IES). Utilized JSF/PrimeFaces 13 and deployed on Payara 6.

- **FiscalBot**: Developed an operational Telegram bot that processes mass ART data and runs scheduled tasks every Monday at 06:00 BRT.

- **Atlas AI**: Built an autonomous system that maintains project documentation using a dual-provider architecture (OpenAI + Claude), capable of routing tasks intelligently based on requirement types.
